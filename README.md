Проект operations_management в рамках практической работы по git

Студентка: Векинцева Виктория Александровна

Группа U4140

**Цель работы:**

Комплексное освоение системы контроля версий Git и платформы GitHub


В рамках работы поставлены следующие конкретные цели:

- Освоение базовых операций Git в локальном репозитории, включая настройку окружения, создание репозитория и работу с коммитами

- Изучение методов хранения и управления исходным кодом, работы с историей изменений проекта

- Приобретение навыков публикации кода и синхронизации с удалённым репозиторием на GitHub

- Освоение процесса разработки через систему Merge Requests и Code Review

- Изучение основ управления правами доступа и настройки безопасности репозитория
